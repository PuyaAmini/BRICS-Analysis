من فرمول محاسبه احتمال شکست نهایی را که در متد `calculate_failure_probability` پیاده‌سازی کردم، گام به گام توضیح می‌دهم:

1. **ابتدا سه ریسک اصلی را محاسبه کردم**:

   ```python
   economic_risk = self.calculate_economic_risk()       # ریسک اقتصادی
   political_risk = self.calculate_political_risk()     # ریسک سیاسی
   governance_risk = self.calculate_governance_risk()   # ریسک حکمرانی
   ```

2. **سپس احتمال نهایی را با این فرمول محاسبه کردم**:
   ```python
   overall_probability = (
       self.weights["Economic_Stability"] * economic_risk +           # 0.30 × ریسک اقتصادی
       self.weights["Geopolitical_Tensions"] * political_risk +      # 0.25 × ریسک سیاسی
       self.weights["Global_Economic_Environment"] * 0.5 +           # 0.20 × 0.5 (ریسک پایه جهانی)
       self.weights["Implementation_and_Governance"] * governance_risk # 0.25 × ریسک حکمرانی
   )
   ```

به عبارت ساده‌تر، من این وزن‌ها را در نظر گرفتم:
- 30% وزن برای ریسک اقتصادی
- 25% وزن برای ریسک سیاسی
- 20% وزن برای ریسک جهانی (که ثابت 0.5 در نظر گرفتم)
- 25% وزن برای ریسک حکمرانی

3. **در نهایت عدد بدست آمده را در 100 ضرب کردم تا به درصد تبدیل شود**:
   ```python
   return overall_probability * 100
   ```

بنابراین، من احتمال نهایی را به صورت یک میانگین وزنی از چهار فاکتور اصلی محاسبه کردم و در نهایت آن را به درصد تبدیل کردم. عدد بزرگتر نشان‌دهنده احتمال شکست بیشتر است.

برای مثال، اگر من این اعداد را داشته باشم:
- ریسک اقتصادی = 0.4
- ریسک سیاسی = 0.6
- ریسک جهانی = 0.5 (ثابت)
- ریسک حکمرانی = 0.3

احتمال نهایی را اینگونه محاسبه می‌کنم:
```
(0.30 × 0.4) + (0.25 × 0.6) + (0.20 × 0.5) + (0.25 × 0.3) = 0.445
```
و پس از تبدیل به درصد:
```
0.445 × 100 = 44.5%
```
